Editing a Project


Overview

The Edit Project screen is displayed as a consequence of clicking the Edit Object button from the Project view page.

The user is presented with the Edit Project screen which allows modifying Project Details.

Two action buttons, Submit and Cancel, are listed at the bottom of the page.


Editing the Project Details

As shown in Figure 12.1, the following fields are available in the Project Details section for editing:

  • Title: *
  • Description: *
  • Lead Institution: *
  • Start Date: *
  • Duration: (expected duration, free form text)
  • Completion Date: *
  • Notes: *
  • ANZSRC Field of Research Code: * Drop down list, select from available values.

Figure 12.1 – Screenshot of Editing Project details

Creator, Creation Date, Last modified by and Last modified date are not user updatable.


Validation

Validation is performed when the user clicks Submit. If any fields are in error then the user is returned to the top of the page. An error box will appear at top of the page describing the nature of the errors and remedial action.
The system confirms the submission to the user by switching to the view tab and displaying the project details page in view (read only mode).
The following additional values will be automatically updated against the Project concrete object:

  • Last Modified by is set as the current user
  • Last Modified on is set to the current date

The user may click Cancel instead of Submit to abandon the Edit Screen and get back to the View Screen.


Updating project participants is the same as in Adding Project Participants during Project creation.

Attaching File References is the same as in Attaching File References during Project creation.
