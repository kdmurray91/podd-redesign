Filling Project Details


Project Details

The Add new Project page requires entering the project details. Figure 10.1.1 is the screen shot of the project details section, it includes the following fields (* denotes required field):

  • Title * : free form text
  • Description : free form text, up to 1024 characters
  • Lead Institution * : free form text field.
  • Start Date * : format YYYY-MM-DD, to be selected from a pop-up Calendar
  • Duration: expected duration, free form text
  • Completion Date : format YYYY-MM-DD, to be selected from a pop-up Calendar
  • Notes: free form text
  • ANZSRC FOR Code * : Field of Research Code, to be selected from a drop-down list

 Figure 10.1.1 – Screenshot of the Project Details on the Add new Project page

The project status and publication status fields are not displayed in the Add new Project screen, upon project object instantiation the default value for project status is Active and publication status is Not Published.

The users needs to fill in all the required fields.

No validation is performed until the Submit button at the end of the page is clicked.
