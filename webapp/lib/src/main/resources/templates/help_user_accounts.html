Managing User's Accounts


Changing User Details

When a registered user is logged in, a Settings link is displayed on the user functions menu bar. The user clicks the link, the User Details screen is displayed as follows.

 

Figure 5.1 – Screenshot of the User Details screen

Through this screen, the users can reset their password or change their profiles by clicking the buttons at the bottom of the page. Click the Edit User button at the bottom of the User Details page, the Edit User Details screen is displayed as Figure 5.2.

 

Figure 5.2 – Screenshot of Edit User Details screen

The user can change personal details and contact details. User name and Email Address can not be changed once registered. Only the administrator can change the users roles. Once the user clicks the Update Details button at the bottom of the page, the system will confirm the update to the user by displaying the updated User Details screen.


Resetting Password

Click the Change Password button at the bottom of the User Details page, the Change User Password screen is displayed (Figure 5.3).

 

Figure 5.3 – Screenshot of Change User Password screen

The user enters the current password in the Old Password field and the new password in the New Password and Confirm New Password fields. Once the Save Password button is hit, the system will confirm the update to the user and the User Details screen is returned.

Note that the password has to be at least 6 characters long with no space. Password entered in the two fields must match.
