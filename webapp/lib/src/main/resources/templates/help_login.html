PODD Login

Registered users of the system need to login to PODD to access their projects and perform functions related to their repository role.

PODD based login is performed using the dialog shown in Figure 4.2.1

Figure 4.2.1: Screenshot of PODD Users Login screen

The user fills in:

  • User ID;
  • Password;
  • (Optional) Select the check box next to the statement “Remember me on this computer”. This means a session is maintained and a user can navigate back to PODD without having to re-establish a connection.

If the user is authenticated, a session is created and the user is taken to the Project Listing screen.

If either the user name or password is not accepted, the user will be redirected to the welcome page. An error message will NOT be displayed. the The user can either try the username and password again or contact the system administrator.
