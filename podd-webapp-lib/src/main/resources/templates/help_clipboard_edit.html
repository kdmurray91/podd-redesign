Editing the Clipboard

The user can edit the Clipboard at any time by clicking on the Clipboard link on the main navigation menu. The Edit Clipboard view is displayed as follows (see figure 16.3.1).

Figure 16.3.1 – The Edit View of the Clipboard

The Edit Clipboard view is a modified view of the Clipboard with the copied predicates / objects listed in the table with the following fields:

  • A selection box to select the object/predicate for deletion. Multiple objects/predicates can be selected at once.
  • An icon representing the PODD type (object, predicate, and file) as displayed in the browser.
  • The Title of the object/predicate in the Clipboard. If it is a predicate then the predicate type is displayed followed by (objects:nn) where nn is the number of direct children objects of this predicate.
  • The Type of the object. Type will not be displayed for predicates on the Clipboard.
  • The Parent of the object/predicate, meaning the title of the parent object (note that predicates will not be displayed, but the actual parent object).
  • The Parent Type or object type of the parent object.

The user may select individual objects to delete by checking the selection boxes and clicking Delete Selected or may delete all objects (despite check boxes) by clicking Delete All.
